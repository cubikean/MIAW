Sélectionner des éléments de la page web et utiliser leurs propriétés

Vous devrez savoir

Sélectionner un groupe d'éléments de la page web à l'aide des différentes fonctions (cf : partie rappel) la fonction document.querySelectorAll est utilisée ici pour les différents exemples.

Par exemple document.querySelectorAll("nav.principal a") permet de sélectionner tous les liens du menu dont la classe est principal.

Entre les parenthèse de la fonction querySelectorAll, on place le paramètre indiquant quelle sélection on souhaite effectuer. Le paramètre est un sélecteur css. On peut donc utiliser document.querySelectorAll(".c1") pour sélectionner tous les élément possédant la classe c1, ou encore document.querySelectorAll("#ex img") pour sélectionner toutes les images incluses dans l'élément dont l'id est ex.

Il est possible de stocker le résultat de cette sélection dans une variable avec let liensDuMenuPrincipal = document.querySelectorAll("nav.principal a"), qui crée la variable liensDuMenuPrincipal et y place les éléments sélectionnés.

Utiliser une propriété propre à un ensemble d'éléments.

Si la variable liensDuMenuPrincipal contient tous les liens du menu principal, alors liensDuMenuPrincipal.length désigne le nombre de liens présents dans cet ensemble. Il est possible de stocker cette valeur dans une variable pour la réutiliser plus tard.

Sélectionner un élément de la page web, utiliser ses méthodes et propriétés

Vous devrez savoir

Sélectionner un élément grâce à son id

Si un élément possède l'id premierArticle, il est possible de le sélectionner grâce à l'instruction document.getElementById('premierArticle').

On peut placer l'élément sélectionné dans une variable afin de le réutiliser plus tard : let lePremierArticle = document.getElementById('premierArticle')

Sélectionner un élément grâce à un sélecteur css

Nous avons vu que querySelectorAll permet de sélectionner plusieurs éléments de la page web.

Par exemple, avec document.querySelectorAll("main img"), je sélectionne toutes les images du main.

Pour sélectionner un seul élément de la page web, nous disponsons de la méthode querySelector, ainsi document.querySelector('main img') permet de sélectionner la première image du main.

Accéder aux propriétés d'un élément

Si la variable lePremierArticle contient un élément de la page web, on peut accéder à la liste de ses classes grâce à lePremierArticle.classList.

Appliquer quelques méthodes de traitement à un élément

Il est possible de modifier la liste des classes d'un élément.

L'ajout d'une classe est réalisable grâce à la méthode add : document.getElementById('premierArticle').classList.add('nouvelleClasse')

La suppression d'une classe est effectuée par la méthode remove : document.getElementById('premierArticle').classList.remove('nomDeClasse')

Accéder aux attributs et au texte d'un élement

Vous devrez savoir

Placer du texte dans un élément

Le contenu textuel d'un élément de la page web est une propriété dont on peut modifier la valeur avec document.getElementById('premierParagraphe').textContent="nouveau contenu".

Récupérer la valeur d'un attribut

Rappel : le texte <img src="images/toto.jpg" alt="la tête de Toto" id="imageToto"> permet d'insérer une image dans une page web.

Ici, src et alt sont des attributs de la balise img.

Le contenu d'un attribut est une information que le programme javascript peut récupérer de la manière suivante : 

         let image = document.getElementById('imageToto') ;
         let nomFichierImage = image.getAttribute('src') ;
         // La variable nomFichierImage contient à présent la valeur "images/toto.jpg"

Modifier la valeur d'un attribut

Un programme peut modifier la valeur d'un attribut gâce à la méthode setAttribute.

Cette méthode nécessite deux paramètres : le nom de l'attribut à modifier, et la valeur que l'on souhaite lui donner.

Par exemple image.setAttribute("src", "images/pifou.jpg") permet de donner la valeur "pifou.jpg" à l'attribut src de l'élément désigné par image.

Gérer les événements crées par l'utilisateur

Vous devrez savoir

Relier un élément, un événement le concernant et une fonction de gestion de l''événement.

Si vous voulez détecter que l'internaute clique sur un élément de la page web, vous devez créer un écouteur d'événement lié à cet élément, qui permettra de déclencher l'exécution d'un sous-programme lorsque le clic surviendra.

Par exemple document.getElementById("m").addEventListener("click",agrandir); indique que lorsqu'on clique sur l'élément dont l'id est m, il faut exécuter la fonction agrandir.

La fonction agrandir est un gestionnaire d'événement, nous devrons écrire cette fonction, par exemple :

        function agrandir(){
            document.getElementById("m").classList.add('grand') ;
            // Ajoute la classe 'grand' à l'élément dont l'id est m
        }

Traiter un ensemble d'éléments

Vous devrez savoir

Parcourir un ensemble

Les selecteurs d'éléments tels que querySelectorAll permettent de récupérer plusieurs éléments de la page web.

Il n'est pas possible de traiter plusieurs éléments à la fois

Ex : si l'on considère ce programme

    // Sélection de tous les liens de menu
    let liens = document.querySelectorAll("nav a") ;
            
    // Tentative de modification des classes
    liens.classList.add('grand') ;

On a un problème avec la fonction add car dans la variable liens contient un ensemble (un tableau) de liens et non un élément unique.

Pour appliquer la fonction add à chaque lien, il va falloir parcourir l'ensemble des liens.

Il existe plusieurs façon d'effectuer ce parcours, par exemple :

        for(n=0; n<liens.length; n++)
        {
            liens.item(n).classList.add('grand') ;
            // ou liens[n].classList.add('grand') ;
        }

La boucle for permet ici d'effectuer plusieurs fois le même traitement.

Ici, on va exécuter un tour de boucle pour chaque élément de l'ensemble liens.

A chaque tour de boucle, on applique l'ajout de classe au nième élément de liens.

Manipuler l'élément cible d'un événement

Vous devrez savoir

Désigner l'élément qui fait l'objet d'un événement

Dans une fonction gestionnaire d'événement, l'élément qui a reçu lévénement est désigné par le mot-clé this.

Par exemple, on veut qu'un clic sur un logo permette de lui attribuer la classe "grand"

          // Sélection de l'élément dont l'id est "logo"
          let leLogo = document.querySelector('#logo') ;
          // Mise en place d'un écouteur pour l'événement "clic"
          leLogo.addEventListener("click", agrandir) ;
          // Création de la fonction de gestion de l'événement
          function agrandir(){
              // Ajouter une classe à l'élément qui a reçu l'événement
              this.classList.add("grand) ;
          }
