Exercices javascript - Suite

Introduction

Voici une série d'exercices d'initiation à JavaScript.

Chaque exercice doit traiter des éléments html compris dans cette page web.

Créez un dossier js. Dans celui-ci, créez un fichier javascript nommé comportements.js que vous lierez à cette page web.

Dans comportements.js, créez le squelette d'un programme javascript, avec l'appel à une fonction principale lorsque le navigateur a fini de recevoir la page web.

Vous structurerez le programme principal et y placerez des commentaires afin que l'on indentifie clairement les instructions de chaque exercice.

Manipulation du DOM

Vous devrez savoir

Supprimer un élément

Pour supprimer un élément, on applique la méthode removeChild à son père.

Cette méthode reçoit en paramètre l'élément enfant à supprimer.

Par exemple, pour supprimer une balise img dans une balise figure :

          // Sélection de l'élément à supprimer
          let image = document.querySelector('#laOuJeVeux figure image') ;
          // Sélection du père
          let figure = image.parentNode ;
          // Ou figure = = document.querySelector('#laOuJeVeux figure') ;
          // Suppression
          figure.removeChild(image) ;

Créer et insérer un élément

La méthode document.createElement permet de créer un nouvel élément à insérer dans la page web.

La méthode appendChild permet d'ajouter un élément à l'intérieur d'un autre, en dernière position.

Par exemple, pour créer une balise img et l'insérer à la fin d'un paragraphe :

          // Création de l'élément
          let nouvelleImage = document.createElement('image') ;
          // Sélection du lieu d'insertion
          let paragrapheDInsertion = document.querySelector('#laOuJeVeux p') ;
          // Insertion
          pere.appendChild(nouvelleImage) ;

Gestion d'évènement et manipulation du DOM

Vous devez savoir :

  • Relier un élément, un évènement le concernant et un gestionnaire d'évènement chargé de traiter l'évènement en question. Par exemple document.getElementById("m").addEventListener("click",traiter); indique que lorsqu'on clique sur l'élément dont l'id est m, il faut exécuter la fonction traiter.
  • Dans un sous-programme de gestion d'un évènement, faire référence à l'élément qui a déclenché l'évènement. Par exemple this.classList.add('penche') ; permet de donner la classe "penche" à l'élement qui a reçu l'événement.
