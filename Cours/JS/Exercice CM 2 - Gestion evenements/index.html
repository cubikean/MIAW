Contexte

Les fichiers html permettent de définir le contenu et la structure des pages web.

Les fichiers css permettent de définir la mise en forme des pages web.

Un programme javascript permet généralement de définir le comportement des éléments d'une page web.

Jusqu'à présent, nous avons vu comment :

  • Créer la base d'un programme javascript avec les variables, les instructions conditionnelles et itératives, les tableaux et objets, et les fonctions.
  • lier un programme javascript à une page web (avec la balise script)
  • en javascript, sélectionner des éléments de la page web (pour les modifier, ou pour les stocker dans une variable et les traiter plus tard)
  • accéder à la liste des classes d'un élément de la page web et modifier cette liste
  • accéder au contenu textuel d'une balise et le modifier
  • accéder aux attributs d'une balise et les modifier

Revoyons rapidement ces points techniques

Mise en place du programme
